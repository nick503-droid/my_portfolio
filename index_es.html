Nutri Time Frontend

Proyecto Académico

Descripción del Proyecto

Construí todo el frontend como una Single Page Application (SPA) interactiva y responsiva utilizando Vue.js. La aplicación consume la API RESTful de Laravel para mostrar recetas, gestionar categorías y autenticar usuarios.

Mis Acciones y Responsabilidades

  • Construcción de todos los componentes de la interfaz de usuario (tarjetas de recetas, formularios, navegación, layouts) con diseño responsivo usando Tailwind CSS
  • Configuración de Axios con un cliente de API centralizado para una comunicación segura con el backend de Laravel
  • Desarrollo del componente de chatbot interactivo "Chef AI" con historial de conversación continuo formateado en Markdown
  • Implementación de lógica de localStorage para persistir mensajes del chatbot y sesiones de usuario entre visitas

Tecnologías Utilizadas

Vue.js Vue Router Pinia Tailwind CSS Axios Node.js

API RESTful de Tareas

Proyecto Académico

Descripción del Proyecto

Construí el backend completo y la API RESTful para la plataforma Nutri Time. La API gestiona toda la lógica de negocio, incluyendo funcionalidad CRUD para recetas y categorías, y un sistema de autenticación de usuarios full-stack basado en Laravel Sanctum. La persistencia de datos se maneja con MySQL.

Mis Acciones y Responsabilidades

  • Diseño e implementación de la arquitectura de la API RESTful y el esquema de base de datos relacional en Laravel
  • Desarrollo y mantenimiento de endpoints CRUD para recetas y categorías utilizando el patrón Modelo–Controlador y Recursos de API para respuestas JSON estructuradas
  • Creación del endpoint "AI Chef" integrando la API de Google Gemini con ingeniería de prompts dinámica y manejo seguro de chat
  • Construcción de un Panel de Administración interno con Blade y Tailwind CSS para una gestión intuitiva de datos sin interacción por línea de comandos
  • Configuración de middleware CORS para la comunicación con el frontend en Vue

Tecnologías Utilizadas

Laravel (PHP) MySQL Laravel Sanctum Google's API (Gemini) XAMPP Node.js

ERP Centro de Reciclaje

Proyecto Freelance

Descripción del Proyecto

Desarrollé una solución ERP personalizada e integral para digitalizar las operaciones de "Recicladora Fernández". El sistema reemplazó el seguimiento manual con entrada digital de materiales, cálculos de costos dinámicos y un módulo crítico de "Cierre de Caja Diario" que detecta variaciones financieras en tiempo real.

Contribuciones Clave

  • Arquitectura de una API REST robusta usando Laravel 11 y un frontend reactivo en Vue 3.
  • Ingeniería del algoritmo de "Conciliación de Efectivo" para comparar ventas del sistema vs. efectivo físico.
  • Implementación de Control de Acceso Basado en Roles (RBAC) seguro para separar privilegios de Administrador y Contador.
  • Integración de generación automatizada de facturas PDF y exportación de datos a Excel.
  • Gestión del entorno de producción, base de datos y despliegue a través de Railway.app.

Tecnologías Utilizadas

Vue.js 3 Laravel 11 Tailwind CSS MySQL Vuetify Railway Axios
